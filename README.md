# Linux Terminal Portfolio Website

This project is a portfolio website designed to look like a Linux terminal.

Users can interact with the website using terminal-style commands to explore the portfolio.

## Features

- Terminal-style user interface
- Command based navigation
- About section
- Projects section
- Contact information
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository
2. Open index.html in your browser

## Commands Example

help        - show available commands  
about       - about the developer  
projects    - list projects  
contact     - contact details
